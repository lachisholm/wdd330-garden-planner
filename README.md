# Garden Planner

Garden Planner is a web application designed to help users plan, design, and manage their gardens. The app provides tools for selecting plants, designing garden layouts, tracking orders, and managing shipping and payment information.

## Features

- **Plant Selection:** Browse and choose from a variety of flowers and vegetables.
- **Garden Design:** Create and customize your own garden layout.
- **Shopping Cart:** Add plants to your cart and manage your selections.
- **Order Tracking:** Track your orders and view shipping information.
- **User Authentication:** Sign in to access personalized features.
- **Responsive Design:** Optimized for both desktop and mobile devices.

## Getting Started

1. **Clone the repository:**
   ```
   git clone https://github.com/yourusername/wdd330-garden-planner.git
   ```
2. **Open the project folder in your code editor.**
3. **Open `index.html` in your browser to start using the app.**

## Customization

- Add your own plant data in the `data/` directory.
- Update images in the `images/` folder.
- Modify styles in `css/styles.css` to change the look and feel.
